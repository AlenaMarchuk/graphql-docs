# GraphQLDocs

Easily generate beautiful documentation from your GraphQL schema.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'graphql-docs'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install graphql-docs
